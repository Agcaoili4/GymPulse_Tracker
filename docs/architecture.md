This contains the architecture of the project and will be further expanded in later phases:

Contains:

- diagrams
- Back-end structure
- Front-end Structure
- Data Flow
- Services

Frontend Structure:

- React with Tailwind CSS
- Typescript for type safety
- Component-based architecture for modularity and reusability
- State management using React Context API for global state

Backend Structure:

- .NET Core Web API
- C# for server-side logic

Data Flow:

- Microsoft SQL Server Docker Container for data storage
- API endpoints for CRUD operations on clubs, occupancy data, user accounts, and reviews
